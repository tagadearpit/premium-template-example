# NOVA Ledger Landing Page

Static HTML/CSS/JavaScript landing page using the supplied hero video.

## Files

- `index.html`
- `styles.css`
- `script.js`
- `assets/coin-running-on-platform.mp4`

## Important

The MP4 asset is copied directly from the uploaded source. It is not re-encoded or recompressed.

## Run locally

Open `index.html` directly, or run:

```bash
python -m http.server 5500
```

Then open:

```text
http://localhost:5500
```
